# ego-planner-ros2-sim
Description of ego planner ros2 simulation in gazebo

## PX4 Firmware Setup
1. Install PX4 via this link: https://docs.px4.io/main/en/dev_setup/getting_started
```bash
make px4_sitl gz_x500
```
Make sure the drone can take off by command
